# 🌍 GeoTrade AI

An AI-powered system that converts real-time geopolitical events into actionable trading insights using LLMs.

## 🚀 Features
- Real-time news ingestion (GDELT)
- Local LLM analysis (Ollama + Llama 3)
- Trading signal generation (Buy/Sell/Hold)

## 🧠 Tech Stack
- Python
- GDELT API
- Ollama (Llama 3)
